# Élan - Frontend

This is the client-side application for **Élan**, a luxury perfume e-commerce platform. It is built using **React** and **Vite**, with a focus on high-performance rendering and premium aesthetics.

## 🚀 Key Technologies
- **React 19**: Modern UI development.
- **Vite**: Ultra-fast build tool and development server.
- **Tailwind CSS**: Utility-first CSS framework for custom, premium styling.
- **Framer Motion**: For cinematic animations and page transitions.
- **Lucide React**: Beautiful, consistent iconography.
- **React Hot Toast**: Elegant notification system.

## 📦 Main Features
- **Responsive Home Page**: Featuring luxury brand storytelling.
- **Product Catalog**: Live search, category filtering, and price sorting.
- **Luxury Product Views**: Detailed specs and high-quality image galleries.
- **Cart Management**: Real-time updates with local storage persistence.
- **Secure Checkout**: Multi-step checkout with form validation.
- **User Dashboard**: Profile editing and order history.
- **Seller/Admin Interface**: Tools for managing the fragrance collection.

## 🛠️ scripts
- `npm run dev`: Starts the Vite development server.
- `npm run build`: Generates the production-ready build in the `dist/` folder.
- `npm run lint`: Runs ESLint for code quality checks.

## 📂 Structure Highlights
- `/src/components`: UI primitives and composite components (Navbar, Footer, ProductCards).
- `/src/pages`: Main application views.
- `/src/context`: Global state (AuthContext, CartContext).
- `/src/assets`: Local images and luxury font definitions.
- `/src/services`: API endpoint calls using Axios.
